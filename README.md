# websiteforsianna
this website is for sianna for any people 
This project is a responsive Arabic website for Junkers Water Heater Maintenance Services in Egypt.
The website provides detailed information about maintenance services, hotline contact (17412), working hours, customer testimonials, and FAQs.

It is built using HTML5, CSS3, Bootstrap 5, and includes:

SEO optimization (Meta tags, Schema Markup)

Responsive design for all devices

Hotline & WhatsApp quick contact buttons

Image slider and service sections

Google Maps integration

Structured FAQ section

The project focuses on user experience, performance, and search engine visibility, making it suitable for real business usage.
